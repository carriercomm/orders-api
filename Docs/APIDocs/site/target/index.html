Order Management API: Introduction

Metida Print Order Management API allows you to place orders to the system and query the statuses of existing orders. It allows Metida Print services to be integrated to your platform easy and smoothly. We provide SDKs for the most popular programming languages:

How to start

Step 1
Get SDK for the service
Step 2
Get your test user credentials for the developer sandbox. This step is optional (you can use the public user test@metidaprint.com with password MetisZeus1450BC) but highly recommendable:
  • Public user data is cleared on a nightly basis (around midnight in Esbo, Finland)
  • It's easier for us to assist you with any inquires you might have if you use dedicated credentials
Step 3
Develop and test your application in the developer sandbox (dev.metidaprint.com:443) with test credentials
Step 4
Get your Metida Credentials
Step 5
Switch your application to use Metida Services (orders.metidaprint.com:443)!

Metida Order Management API: how to start?

The ultimate API design goal for us is simpicity. We made Order Management API as simple as we could. Judge yourself:
  1. Authorize against the service using the method getAuthToken
  2. Fill the order details using the structures ProductData, ShipmentData and OrderMiscDetails
  3. Place your order by calling the method newOrder
As simple as that!
